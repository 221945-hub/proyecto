# Sistema Web de Gestión de Pedidos de Comida Local

## 📌 Descripción del Proyecto
Este proyecto consiste en el desarrollo de un sistema web para una startup ubicada en Cusco, Perú, dedicada a la gestión de pedidos de comida local.  

El objetivo es permitir que los clientes puedan realizar pedidos en línea de manera rápida y sencilla, mientras que el negocio administra productos, pedidos y clientes desde una plataforma centralizada.

---

## 🎯 Objetivos
- Facilitar la gestión de pedidos de comida local.
- Organizar el código del proyecto utilizando Git y GitHub.
- Implementar control de versiones desde el inicio del desarrollo.
- Mejorar el trabajo colaborativo del equipo.

---

## 🛠️ Tecnologías Utilizadas
- Git
- GitHub
- Linux Ubuntu
- HTML / CSS / JavaScript
- Backend Web (por definir)
